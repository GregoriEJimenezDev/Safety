Manual de usuario - SafeMap RD

Esta guia te ayuda a usar la aplicacion para reportar incidentes y evitar zonas de riesgo.

1. Ver incidentes recientes

  • Usa el filtro Tipo para ver atracos, asaltos o robos.
  • Usa Periodo para limitar a 7, 30, 90 dias o todo.
  • Haz clic en una tarjeta para centrar el mapa en ese incidente.

2. Reportar un nuevo incidente

  1. Haz clic en el mapa para seleccionar coordenadas.
  2. Completa descripcion, sector y fecha.
  3. Presiona Guardar reporte.
El formulario valida datos para prevenir errores: descripcion minima, coordenadas validas en RD y fecha no futura.

3. Zonas calientes

La seccion "Zonas calientes" agrupa reportes por sector y muestra nivel de riesgo por color.

4. Panel administrador

  • Inicia sesion para habilitar eliminacion de reportes falsos.
  • Al eliminar, puedes usar Deshacer durante 10 segundos.

5. Atajos

  • ? Abrir ayuda
  • Alt + R Actualizar datos
  • Esc Cerrar ayuda

6. Solucion de problemas

  • Si ves "Sin conexion", verifica internet o estado del backend.
  • Si no puedes eliminar incidentes, confirma sesion admin activa.
  • Si falla el reporte, revisa mensaje de validacion mostrado en rojo.
